# M4-A4 行情工作台

## 范围

M4-A4 提供手工行情商品和历史报价的操作工作台：

- `/market`：行情商品列表，支持 URL 筛选、分页、移动端卡片和桌面表格。
- `/market/[marketItemId]`：行情商品详情、当前有效行情、历史报价和生命周期操作。
- 只使用现有 `/api/market/*` API；页面不直接访问 Prisma。

## 页面规则

- 全部平台、报价类型和生命周期状态使用中文显示。
- 列表仅展示 API 派生的当前行情；没有当前行情时引导用户到详情查看 `availabilityReason`。
- 详情展示服务端返回的标准化名称和标准化 SKU，标准化字段不允许由页面提交。
- 新建、编辑、确认、失效、替代修正、停用和重新启用均以 API 的 `availableActions` 为页面入口依据。
- 失败时保留用户已填写内容，不做乐观更新；成功后重新读取 API 数据。

## 生命周期边界

- 报价创建默认未确认；只有服务端确认且未失效、未过期的 Quote 才可能成为当前有效行情。
- 修正通过“失效原 Quote + 新建替代 Quote”完成，历史记录不覆盖也不删除。
- 已停用的行情商品保留历史，不能新增或确认报价；允许失效既有报价，重新启用后才可继续录入。

## 明确不做

- 不计算预计利润、最高采购价或平台推荐。
- 不创建采购订单，不修改采购成本、库存、销售、售后或 `SOLD`。
- 不自动采集平台数据，不接入通知，不新增图表。

M4-A4 页面工作台完成后，M4-A 仍处于进行中状态；采购决策计算属于后续独立实施阶段。
